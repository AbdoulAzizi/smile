KModes

Related Doc: package clustering

class KModes extends PartitionClustering[Array[Int]] with Serializable

K-Modes clustering. K-Modes is the binary equivalent for K-Means. The mean update for centroids is replace by the mode one which is a majority vote among element of each cluster.

Complexity is O(k.t.n) with Hamming distance, quadratic with other metrics.

References

  • Joshua Zhexue Huang. Clustering Categorical Data with k-Modes. http://www.irma-international.org/viewtitle/10828/
  • Antara Prakash, et al. Review on K-Mode Clustering. https://www.ijecs.in/index.php/ijecs/article/download/3058/2834/
  • Python implementations of the k-modes and k-prototypes clustering algorithms for clustering categorical data. https://pypi.python.org/pypi/kmodes/
